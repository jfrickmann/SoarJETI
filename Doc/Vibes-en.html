Vibes

Description

This app allows you to setup event control of the haptic vibrator in your DS-12 transmitter. It is similar to the Sounds on Events menu.

Usage

Press F3(+) to add a new control. Then select a switch for the event and the vibration profile. Optionally, set a delay before the first signal and a delay between repetitions.

You can download the source code from the SoarJETI GitHub repository.
